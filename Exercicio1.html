Utilize a Tag Header

Usando o Header

O header é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML. Nele podemos inserir elementos de tag h1 a h6, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.

Utilize a Tag article

Usando o Article

Seção 1

Utilizamos o elemento article quando precisamos declarar um conteúdo que não precisa de outro para fazer sentido em um documento HTML, por exemplo, um artigo em um blog. É recomendado identificar cada article com um título.

Utilize a Tag section

Usando o Section

O elemento section representa uma seção dentro de um documento e geralmente contém um título, o qual é definido por meio de um dos elementos entre h1 e hë. Podemos utilizar o section, por exemplo, para descrever as seções/tópicos de um documento.

Utilize a Tag nav

Usando o nav

O elemento nav é utilizado quando precisamos representar um agrupamento de links de navegação, que, por sua vez, são criados com os elementos ul, li ea.

Utilize a Tag aside

Usando o Aside

O elemento aside é utilizado quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal. Por exemplo, ao falar de HTML semántico, podemos indicar ao leitor outros conteúdos sobre a linguagem HTML como sugestão de leitura complementar.

Utilize a Tag div

A tag é uma das tags mais utilizadas no HTML e é essencial para criar layouts e estruturas em um site. Ela é utilizada para dividir o conteúdo em seções lógicas e organizá-las de forma visualmente agradável.
